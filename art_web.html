In 2023, I developed a new intention of making "Internet art." I consider this to be goofy, inner-child type works that are made in old abandonware software from the late '90s and early '00s. My ethos is that every piece of software can be a formal art program if you have low enough standards and minimal ego. :) I have linked every program I used for these works below; all of them are free to use or abandonware.

Barbie Sticker Designer

My wife and I crafted these beautiful Barbie memes in this PC game from 1999.




More will be reuploaded soon where I used the following programs:

  • LibreCalc - The open-source version of M*crosoft Excel.
  • JSKidPix - A web-based version of OG Kid Pix written in JavaScript. Really cool stuff.
  • The following Kid Pix versions: Kid Pix Studio Deluxe 3, Kid Pix Deluxe 3, Kid Pix Deluxe 4
  • My Scene: Room Makeover - This flash game was an absolute banger at sleepovers.
  • Barbie Storymaker - PC game from 1996.
  • Nickelodeon 3D Movie Maker - One of the most confusing-ass interfaces ever, but damn does it have so many FEATURES.
